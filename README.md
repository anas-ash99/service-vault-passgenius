# Vault Service - PassGenius

Vault service for PassGenius, securely storing user passwords and secrets.

## Architecture

This service is built using **Java 21** and **Spring Boot 3.3.4**.

### Tech Stack

*   **Framework**: Spring Boot 3.3.4 (Web, Data MongoDB, Test)
*   **Language**: Java 21
*   **Database**: MongoDB

### Key Features

*   Secure storage of user secrets.
*   CRUD operations for vault items.
